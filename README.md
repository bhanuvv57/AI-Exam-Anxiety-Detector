# AI Exam Anxiety Detector

An AI-based system that detects exam anxiety levels from student text inputs.

## Features
- Detects anxiety level from text
- Classifies into:
  - Low Anxiety
  - Moderate Anxiety
  - High Anxiety
- Provides helpful tips to manage exam stress
- Simple interactive UI built with Streamlit

## Technologies Used
- Python
- Streamlit
- Natural Language Processing
- Machine Learning

## Installation

Clone the repository:

git clone https://github.com/bhanuvv57/AI-Exam-Anxiety-Detector.git

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

## Example Input
"I am very nervous about my exam tomorrow"

## Output
Moderate Anxiety
